# Traceclip (vibecheck) 🧼

**Traceclip** is a fast, local code sanitizer and linter designed specifically to prune and inspect AI-generated code. 

Rather than enforcing pedantic style formats like traditional linters, `vibecheck` hunts down common AI developer artifacts and "hallucinations":
- 🚫 **Unused Imports**: Bloat imported by AI but never referenced.
- 🪦 **Dead Functions**: Functions generated by the AI that are defined but never called or referenced.
- 🔑 **Placeholder Variables & Secrets**: Hardcoded templates like `API_KEY = "your_key_here"` or `URL = "https://example.com/api"`.
- 🕳️ **Stub Implementations**: Empty functions with only `pass`, `...`, or `raise NotImplementedError`.

It grades your script with a **Vibe Score** (out of 100). If your score drops too low, it fails!

---

## Installation

Inside the project directory, run:
```bash
pip install -e .
```

## Usage

Check a script:
```bash
vibecheck my_script.py
```

Check multiple files or directories:
```bash
vibecheck src/ tests/
```

Automatically sanitize and fix issues:
```bash
vibecheck my_script.py --fix
```

Enforce a minimum vibe score (ideal for CI/CD):
```bash
vibecheck my_script.py --min-score 80
```
